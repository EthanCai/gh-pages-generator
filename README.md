# 说明

- `404.html`仅在网站绑定顶级域名下才起作用

# 常用操作

```
> # Create draft
> hexo new draft "a new blog"

> # Start local server, view blog including draft
> hexo server --draft

> # Publish draft
> hexo publish "a new blog"

> # Start local server, view blog not including draft
> hexo server

> # Generate static files
> hexo generate

> # Deploy website
> hexo deploy
```
